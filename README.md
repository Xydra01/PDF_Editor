# PDF Editor

  ## Description

  This project is a dedicated file to PDF converter and editor that allows the user to convert files to PDF format as well as merge, split, and edit them.

  ## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

Install all dependencies via pip install (figuring this out).

## Usage

Convert non PDF files into PDF format. Merge PDFs and even non PDFs into a single PDF. Seperate PDF into multiple PDF Files. Add or remove pages from PDFs. Edit or remove text from a PDF.
